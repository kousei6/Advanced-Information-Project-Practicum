# app.py

**機能概要: 呼吸するソフトウェアキーボード (Dynamic Breathing Keyboard)**

1.  **動的な伸縮アニメーション**
    * CSSアニメーションを使用し、キーボード全体が「呼吸」するように周期的に拡大・縮小します。
2.  **アニメーション設定のカスタマイズ**
    * サイドバーのスライダーを使用し、以下のパラメータをリアルタイムに変更できます。
        * **呼吸の速さ:** アニメーションの周期（0.1秒〜5.0秒）。
        * **最小サイズ:** 縮小時の倍率（0.5倍〜1.0倍）。
        * **最大サイズ:** 拡大時の倍率（1.0倍〜1.5倍）。
3.  **インタラクティブな文字入力**
    * 画面上のキーをタップ（クリック）することで、上部のテキストエリアに文字を入力できます。
    * BackSpace (BS) キーによる文字削除も可能です。
4.  **ワイド＆レスポンシブレイアウト**
    * Streamlitの標準余白をCSSで削除し、画面の横幅を最大限（100%）まで使用して表示します。
    * Flexboxを使用しており、画面サイズに合わせてキーの幅が自動調整されます。
5.  **キー配列とデザイン**
    * 標準的なキーボード配列（英字・かな併記）を再現しています。
    * 特定のキー（Enter, Shift, 特殊キーなど）が色分けされており、視認性を高めています。

# app1.py
**機能概要: 浮遊・呼吸するソフトウェアキーボード (Floating & Breathing Keyboard)**

1.  **二重アニメーション（呼吸＋浮遊）**
    * **呼吸:** キーボード全体が周期的に拡大・縮小します（`scale`）。
    * **浮遊:** キーボード全体が画面内を上下左右に不規則に移動します（`translate`）。
    * これら2つの動きを個別のCSSラッパー（`.movement-wrapper` と `.keyboard-wrapper`）で管理することで、干渉せずに同時に実行しています。
2.  **詳細なパラメータ設定**
    * サイドバーで以下の設定を独立して調整可能です。
        * **呼吸:** 速さ、最小サイズ、最大サイズ。
        * **浮遊:** 移動の速さ、移動距離（動きの激しさ）。
3.  **インタラクティブな入力**
    * 動き回るキーボードをタップ（クリック）して、固定表示された入力欄に文字を打ち込むことができます。
4.  **ワイドレイアウト**
    * 画面幅を100%使用し、レスポンシブに表示されます。
